Defined in: ..\js\assets\modules\service\layer\queriers\QuerierPanelService.js:1

registerPanel

(
  • openPanelFn
  • setXMLFn
)

Defined in ..\js\assets\modules\service\layer\queriers\QuerierPanelService.js:103

Register the panel by passing in some querier panel controller functions.

Parameters:

  • openPanelFn Object

    used to open and close the panel function openPanelFn(ctrlBool, useApply) 'ctrlBool' if true panel will open, if false panel will close 'useApply' boolean parameter. Set to true and 'openPanelFn()' will use the '$apply()' method to make the panel open. Set to false and '$apply()' will not be used. It is recommended to set to false in places where you would get an '$digest already in progress' error, e.g. calling 'setPanel(true)' from within a 'then()' function

  • setXMLFn Object

    function used to display XML data in the panel function setXMLFn(xmlString, layerName, prependStr) return boolean 'xmlString' is a string of xml to be displayed 'layerName' layer display name 'prependStr' string to prepend to layer display name returns true is there is data to display
