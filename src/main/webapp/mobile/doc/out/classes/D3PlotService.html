(

Provided by the controllers module.

Defined in ..\js\assets\controllers\querier\querierPanelCtrl.js:52

Used to display XML in tree form in the panel, calls parse() function Passed to 'QuerierPanelService' as a parameter, then called by 'QuerierPanelService' It is possible for node to be passed in as a feature node or the whole collection. We attempt to be smart by assuming it is a feature collection and if it fails, we assume it to be a feature node.

Parameters:

  • xmlString Object

    string which will be displayed

  • displayName Object

    name of layer or feature, to be used only if no suitable name is found within XML string

  • appendFlag Object

    will append the new tree to the current tree(s) on panel or clear the panel and add a new tree

Returns:

boolean value, true if the panel should be opened because there is something to display
